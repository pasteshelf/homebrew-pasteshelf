# homebrew-pasteshelf

Homebrew tap for [PasteShelf](https://github.com/pasteshelf/PasteShelf) — privacy-first clipboard manager for macOS.

## Install

```bash
brew tap pasteshelf/pasteshelf
brew install --cask pasteshelf
```

## Update

```bash
brew upgrade --cask pasteshelf
```

## Uninstall

```bash
brew uninstall --cask pasteshelf
```
